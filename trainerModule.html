Object Oriented Programming

Exam Preview

  1. List the four main features of Object-Oriented Programming.
    2. Inheritance, Encapsulation, Polymorphism, and Data Abstraction.
  2. What is a class?
    3. A class is a prototype that consists of objects in different states and with different behaviors. It has a number of methods that are common the objects present within that class.
  3. What is an object?
    4. An object is a real-world entity which is the basic unit of OOPs for example chair, cat, dog, etc. Different objects have different states or attributes, and behaviors.
  4. What is hybrid inheritance?
    5. Hybrid inheritance is a combination of multiple and multi-level inheritance.
